Common fields for the Files resource include:
Metadata:
 id: The unique identifier of the file or folder.
 name: The name of the file or folder.
 mimeType: The MIME type of the file.
 size: The size of the file in bytes.
 createdTime: The creation timestamp of the file.
 modifiedTime: The last modification timestamp of the file.
 parents: A list of parent folder IDs.
 trashed: A boolean indicating if the file is in the trash.
 explicitlyTrashed: A boolean indicating if the file was explicitly trashed by the user.
 capabilities: Capabilities of the current user on this file (e.g., canEdit, canShare).

Permissions and Sharing:
 permissions: A list of permissions for the file, including role, type, and email address.
 sharedWithMeTime: The time the item was shared with the current user.
 sharingUser: The user who shared the item.

Content-specific metadata (for certain file types):
 imageMediaMetadata: Metadata for images (e.g., dimensions, camera make/model).
 videoMediaMetadata: Metadata for videos (e.g., duration, dimensions).

Other useful fields:
 webViewLink: A link to view the file in a web browser.
 webContentLink: A link to download the file content.
 thumbnailLink: A link to a thumbnail image of the file.
